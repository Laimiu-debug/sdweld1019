🚀 首次部署（3步）
-----------------

步骤1: 配置QQ邮箱授权码
   编辑文件: backend\.env.production
   找到: SMTP_PASSWORD=你的QQ邮箱授权码
   替换为你的真实授权码（16位字符）

步骤2: 推送代码到Git
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin <你的仓库地址>
   git push -u origin main

步骤3: 运行部署脚本
   .\首次部署到服务器.ps1
